What does it mean to display inline vs inline blocks?

To understand the difference between  display: inline  and  display: inline-block  we must first understand the terms "block" and "inline".
(block, inline, and inline-block are all values of the "display" attribute)

block elements are always displayed on a new line and contain a margin on the top and bottom. They will also take up the full available width of the parent element. Some common default block elements are <p>, <div>, and <h1> to <h6>.

inline elements are displayed on the same line as the element before them and will only take up as much space as the content they contain. Some common default inline elements are <span>, <a>, and <code>.
