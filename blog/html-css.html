What does it mean to display inline vs inline blocks?

To understand the difference between  display: inline  and  display: inline-block  we must first understand the terms "block" and "inline".
(block, inline, and inline-block are all values of the "display" attribute)

block elements are always displayed on a new line from the previous element and contain a margin on the top and bottom. They will also take up the full available width of the parent element.
